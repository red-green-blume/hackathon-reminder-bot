# hackathon-reminder-bot
Бот с командными играми для изучения английского.

## Игры
### Шпион
Нужно как минимум три игрока. После старта один игрок получает в личные сообщения роль шпион, остальные
получают локацию, в которой они все находятся. Задача шпиона узнать локацию, задача остальных найти вычислить 
шпиона. Для этого на каждом ходу игроки задают друг другу вопросы. Вопросы должны быть на английском языке, а
за использование слов из заданного набора участники получают дополнительные баллы. 

### Скоростной перевод
Бот пишет слово на английском, участники наперегонки их переводят, в конце получаем топ самых быстрых знатоков
~~на диком западе~~.

### Слова
Всем знакомая и простая игра в слова.

## Запуск
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
BOT_TOKEN='' python3 master_bot.py
```
